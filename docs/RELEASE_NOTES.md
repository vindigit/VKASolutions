# Release Notes

## 0.1.0 - 2026-05-13

Initial VKA CRM MVP release.

### Added

- Next.js application shell
- Fake login flow with seeded demo users
- Dashboard with scenario and account summaries
- Customer list and customer detail pages
- Report IT Issue form
- IT admin tools page
- Role-based access simulation
- Documentation for project scope, workflows, scenarios, and user accounts

### Notes

- All data is local and seeded in code
- No real authentication is used
- No paid or external service is required for the app
