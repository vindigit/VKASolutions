# VKA CRM

VKA CRM is a simulated internal CRM for the fake company VKA Solutions. It is used as a help desk training environment for beginner IT support technicians.

## Purpose

This app creates realistic IT support scenarios such as:

- Account lockouts
- Password resets
- CRM access issues
- Duplicate customer records
- Application errors
- Role-based permission problems

## Users

- Vindon Alexander — CEO
- Brother 1 — Tier 1 Support Technician
- Brother 2 — Tier 2 Support Technician

## Support Workflow

1. User experiences issue in VKA CRM.
2. User submits ticket through Freshdesk.
3. Technician receives ticket notification.
4. Technician troubleshoots issue.
5. Technician documents steps taken.
6. Technician closes ticket.

## MVP Features

- Fake login
- Dashboard
- Customer records
- Role-based access
- Report IT Issue form
- Admin tools for IT support
