# AGENTS.md

## Project
VKA CRM is a fake internal CRM for VKA Solutions, used to train beginner IT support technicians.

## Goal
Build a simple CRM that creates realistic help desk scenarios involving login issues, permissions, customer records, and application errors.

## Tech Stack
- React or Next.js
- JavaScript or TypeScript
- Simple seeded data
- No paid services required for MVP

## Rules
- Keep the app simple.
- Do not add complex authentication.
- Use seeded fake users.
- Use clean, readable code.
- Document setup steps in README.
- Every feature should support IT help desk training.

- Login page

- Role-based access simulation
- Report IT Issue form
- IT admin tools page
